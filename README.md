## 📖 프로젝트 개요
LotteOn 프로젝트는 **롯데 e-커머스** 환경과 유사한 **쇼핑몰 플랫폼**을 개발하기 위해 설계되었습니다. 쇼핑몰의 기본적인 기능을 포함하며, 사용자 경험과 효율적인 백엔드 설계에 초점을 맞춘 웹 애플리케이션입니다. **SpringBoot**와 **Thymeleaf**, **JPA**를 이용하여 개발되었습니다.

## 👩‍💻 내가 맡은 기능
### 📍 전반적인 회원 기능 
1. **구매자 & 판매자 회원가입 / 로그인**
    - 회원 유형(구매자, 판매자)에 따른 회원가입 및 로그인 구현
    - 입력값 유효성 검사 및 예외 처리
    - **Spring Security 기반 세션 인증 적용**
2. **추가 로그인 기능**
    - **소셜 로그인** : **OAuth 2.0**을 활용한 카카오, 네이버, 구글 로그인 지원
    - **자동 로그인** : **Remember-Me** 기능을 활용한 로그인 유지
3. **인가 설정**
    - 판매자로 로그인 시 관리자 페이지로 자동 이동
    - 관리자, 구매자, 판매자별 접근 권한 설정 (인가 처리)
    - 로그인이 필요한 서비스에 대한 접근 제어 설정
4. **아이디 / 비밀번호 찾기**
    - **SMTP 이메일 인증**을 활용한 계정 정보 찾기 기능

1. **나의 설정 페이지**
    - 개인정보 수정 (닉네임, 이메일, 비밀번호, 주소 등)
    - 회원 탈퇴 처리


1. **관리자 회원 관리**
    - 회원 목록 조회 및 검색 기능
    - 회원 등급 선택 및 수정


### 📍 장바구니 기능

1. **상품 페이지에서 장바구니 추가 / 구매 기능**
    - 선택한 상품을 장바구니에 추가하고 구매 가능
    - 상품 옵션 변경 가능
    - 결제 금액 계산
    - **JavaScript**를 활용한 실시간 가격 계산 기능
    


### 📍 약관 기능

1. **약관**
- 구매자&판매자에 따른 약관 출력
- 관리자 약관관리 기능
